user_options(other_user_id) → None¶

If the user option is selected, this function receives the selected users ID, - displays the user options (STATS FOLLOW TWEETS RETURN) - gets user selection - IF STATS, call Controller.display_user_stats(other_user_id) - IF FOLLOW, call Controller.follow_user(self.main_user_id, other_user_id) - IF TWEETS call Controller.users_tweets(other_user_id) to get a list of tweets from other user then call UserInterface.user_options_show_tweets(twt_list, other_user_id) - IF RETURN, call UserInterface.menu()

Parameters:

other_user_id (int) – the user ID of the user that the options are for

Returns:

None

user_options_show_tweets(other_user_id) → None¶

IF twt_list is empty, display message and return to user_options ELSE display up to 3 tweet by other_user. IF there are more then 3 tweets in twt-list, prompt to show more Go back to user_options

Parameters:

  • twt_list (list[int]) – list of tweet ID’s

  • other_user_id (int) – the user ID of the user that the options are for

Returns:

None
